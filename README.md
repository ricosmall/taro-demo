# A simple demo of taro

类 React 语法写一份代码，生成微信小程序、H5页面、快应用、React Native 应用等。

## Usage

1.全局安装 taro

```sh
npm install -g @tarojs/cli

# 可以不装，npm 5.2 以上版本可以直接使用 npx
```

2.初始化一个 taro 项目

```sh
taro init your-app

# or

npx @tarojs/cli init your-app
```

3.开发

```sh
# write your code
```

4.小程序模式实时预览

```sh
npm run dev:weapp
```

5.H5 实时预览模式

```sh
npm run dev:h5
```
